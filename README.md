# ZYNQ Project (UN)

ZYNQ turns scattered games into a single living system.

## Collection

500+ open-source games in one structured ecosystem for building, exploring, and reusing.

## Structure

* g1 – shard
* g2 – shard
* g3 – shard
* g4 – shard
* g5 – shard
* thumbnails – assets

All shards are equal parts of the same system.

## Contributing

Pull requests are welcome on any shard repository.

Improve, fix, or expand the system.

## Purpose

Make game creation faster, cleaner, and shared.
