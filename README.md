# ZYNQ Project (UN)

## What is it?
The ZYNQ Project by HyperRush hosts 500+ open-source games, all collected with metadata to help anyone build their own gamesite.  
It also provides a library to make this process easier for developers.

## When was it started?
The project began in early April 2026.

## How can I contribute?
You can contribute by opening a pull request on any of the `g(X)` repositories, which hold the games.

## Repositories
- **g1** – Game shard 1  
- **g2** – Game shard 2  
- **g3** – Game shard 3 (primary games collection)  
- **g4** – Game shard 4  
- **g5** – Game shard 5  
- **thumbnails** – Game thumbnails
