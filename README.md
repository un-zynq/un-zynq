# ZYNQ Project (UN)

## What is it?

The ZYNQ Project by HyperRush is a large open-source collection of 500+ games gathered into one structured ecosystem.

Its goal is simple:
Make it easy for developers and creators to build, study, and launch their own game sites using ready-made game resources.

It also includes a lightweight library system that helps organize and load games efficiently across projects.

---

## Why does this project exist?

We believe game creation should be:

* Fast to start
* Easy to build on
* Open for everyone

So instead of games being scattered across random sites, ZYNQ brings them together into one structured collection.

---

## When was it started?

The project began in early April 2026 and is actively expanding.

---

## Repository Structure

Each repository contains part of the full game collection:

* g1 – Game shard 1
* g2 – Game shard 2
* g3 – Game shard 3
* g4 – Game shard 4
* g5 – Game shard 5
* thumbnails – Game preview images

All shards are equal parts of the collection.

## How can you contribute?

We welcome contributions to improve the project.

You can help by:

* Opening pull requests on any g(X) repository
* Improving metadata or structure
* Adding or improving thumbnails
* Reporting broken or missing games

Every contribution helps improve the ecosystem.

## What is this project really about?

ZYNQ is not just a collection. It is a shared workspace for developers.

It allows people to:

* Explore games
* Reuse ideas
* Build new experiences
